## 概要

このサイト自体です。Next.jsのApp Routerを使い、プロフィールと作品一覧をローカルのJSON/Markdownファイルから読み込んで表示しています。

## 使った技術

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- gray-matter / remark（Markdownのパース）
